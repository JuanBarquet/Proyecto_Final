# Objetivo del sistema
Desarrollar un juego interactivo por consola que permita a un usuario competir contra la computadora en el clásico "Piedra, Papel o Tijera". El proyecto busca aplicar y consolidar conceptos fundamentales de programación en Python, tales como el uso de estructuras de datos (listas, tuplas y diccionarios), flujos de control (bucles while y for), creación de funciones y manejo de entradas del usuario.

## Descripción de funcionalidades
El programa cuenta con las siguientes características:
**Entrada de usuario interactiva:** Permite al jugador escribir su elección, validando que sea una opción correcta y tolerando el uso de mayúsculas o minúsculas.
**Oponente virtual (IA básica):** Utiliza la librería `random` para que la computadora escoja su jugada de forma completamente aleatoria y justa.
**Motor de reglas optimizado:** Evalúa el resultado de cada ronda (victoria, derrota o empate) utilizando un diccionario de mapeo, evitando el uso excesivo de condicionales y haciendo el código más eficiente.
**Ciclo de juego continuo:** El usuario puede jugar múltiples rondas de forma ininterrumpida hasta que escriba la palabra "salir".
**Historial de partidas:** El sistema almacena el resultado de cada ronda en una lista y, al finalizar el juego, imprime un resumen detallado de todos los enfrentamientos.
